# Styling & Design System

## 1. Aesthetic Directive
- **Style:** High-density, monochrome terminal aesthetic.
- **Inspiration:** Retro-futurism, command-line interfaces, rigid grids.
- **Colors:** Use a curated palette (e.g., Cyberpunk Dark or Forensic White). No generic SaaS blue.

## 2. Typography
- **Fonts:** Monospaced (e.g., JetBrains Mono, Fira Code) for technical data. Clean san-serif (e.g., Inter, Roboto) for narrative content.

## 3. UI Components
- **Buttons:** Sharp corners, high contrast on hover.
- **Grids:** Use 8px base grid.
- **Animations:** Subtle micro-interactions. Zero bounce.

## 4. Forbidden
- **No Placeholders:** All UI state must be intentional.
- **No Over-Engineering:** YAGNI applies to CSS too.
